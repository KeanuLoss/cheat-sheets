Usage of ordererd lists  <ol>

By using ordered lists you neither need to use breaks, nor use spaces to create indents.
Since the lists aren't inside a paragraph, the line spacings are smaller.
If bigger line spacings are needed, you can put the list within a paragraph element.

Notice:
  • you need to create a paragraph for every line that you want a bigger line spacing for.
  • Every line has to be within an <li> element

absolute references

By using anchor elements <a> you can reference a link to a word.


If we use the abbreviation from the previous example, we can link an article about LLMs to the abbrevation.
The anchor element needs to be placed within the abbrevation element.
Since the source is not on the same server as your html file, an absolute reference (a whole URL) is needed instead of a relative reference.


With the integration of LLMs like ChatGPT or llama you can build software that can do complex tasks which wouldn't be possible without human interaction.

targets

With the target="" attribute you can change how the link is opened:
_blank
opens link in a new tab
blank
opens link in a new tab BUT if you use multiple links with a blank target, they all open in the first new tab instead of creating a new tab for every link
_self
opens link in same tab (default)
_parent
_top

image links

When you want to use an image as a link, all you have to do is use an image-element instead of text within the anchor tag.

relative reference

When you want to create a reference to a page that's on your server, you use a relative reference.

In order to do this you use an anchor element as well, but for the reference you use the html file in the folder where your index file is located.

navigation

By using section elements <section> around different sections within your website you can create a navigation.

In order to do so you have to do these steps:

  1. create sections and give them id's <section id="...">
  2. create a nagivation with <nav>
  3. create an unorganized list with list items for every section id

If you use just an # as the reference in your anchor tag you can create a "back to top"-function

download links

You can create download-links by creating an anchor tag <a> by setting the file name, which needs to be within the folder of the html file, as the reference, followed by the download attribute.

This is what the code looks like
<a href="file name"download>what will be downloaded</a>

adding images

You can add images by using image elements <img>.
For the source you need to enter the folder/file name, the "alt" displays the text, in case the file can't be loaded.

You can adjust the width and height by adding width="..." or height="..." attribute or using a style attribute <style="width:...px; height:...px;">
If you don't do this, the page might flicker while images load.


You should make it match to the actual size of the used image, in order to avoid cummulative shift.

By changing the loading attribute you can enhance the performance of your website by changing it from the default setting "eager" to "lazy".


background image

If you want to use an image as a background, you need to use a style attribute within the body tag.
<body style="background-image:url('file name');">

If you want an image for a background only for a certain paragraph, all you need to do is use a style attribute within the paragraph tag.

adding videos

You can embed and display videos on your website by using a video element <video>.

In order to play the video you need to use a controls attribute.

By using the width="" attribute you can change the size.

By using a poster="" attribute you can display an image as a thumbnail for your video.

By using the attribute autoplay the video immediately starts playing.

By using iframe elements <iframe> you can display the content of another website within your own website.
If you want to show a youtube video on your website you can simply copy the iframe-code from youtubes "embed-menu".

image captions

If you want to add a caption to your images, you need to take two steps:
  1. create a figure tag around the image
  2. create a figcaption underneath the image

You can also add a paragraph within the caption, in order to do so you have to create a figure and within that figure you add a figcaption, which contains the paragraph.

semantic-elements

Semantic elements are very important to structure your website.
They make it easier to read & scan the HTML, improve the SEO as well as the accessability with e.g. screen readers.

A website consists of three parts:
1. the header
contains the h1 and the navigation
2. the main element
contains all of the sections of your website
3. the footer
contains navigations as well

You do so by using an aria-label <aria-label>="primary/secondary navigation"

tables

By using table elements you can create tables.
After creating a table element <table> you need to create a table row <tr>.
Inside the table row you need to create a table data element <td>. Each data element creates a column.
When you create a table, you put the different contents in individual table data elements.

By using table header elements <th> you can create table headers. These need to be placed within table rows as well. They will have a stronger font.

By using rowspan="2" within a table data you create a row that is split in two. <td rowspan="2">...</td>

Using colspan="2" within a table data you create a row that takes up the width of both columns. <td colspan="2">

Example:
business hours

8am - 12:30pm business open
12:30pm - 1pm business closed
1pm - 5:30pm business open
outside business opening hours available via phone
available via e-mail
Contact us to create a better world together
contact us

let us know how we can help you
terms of conditions

headings in rows

If you want to create headings for individual rows, you do so by creating a column with a table head element <th>.
By using table headings you can use scopes to optimize semantics.
If the heading is for a row you need to use a scope="row", for a column you need to use scope="col".
<th scope="row/col">

forms

form elements

By using form elements you are able to create forms.
You need to define two attributes:
1. action
where the form-data will be send to
2. method
specifies how the form-data is send
get: form-data in URL -> never us for sensitive data!
post: form-data inside the body of HTTP request(not shown in URL)

form label

The content of forms are usually inputs, these need to be labeled first. The for attribute value needs to match the id of the input.

form input

For the form input several attributes need to be defined:
1. type
depends on what information is to be entered
2. name
how the information is identified at the server
3. id
needs to match the attribute value of the label
4. placeholder
displays a placeholder until information is put in
5. autocomplete
shows previously input information if turned on
6. required
makes an input required in order to send the form
7. autofocus
focuses the cursor into the field when the website is loaded

dropdown option

If you want to create a dropdown option where you can choose from different options you need to use a select element <select>
Within that element you need to create option elements.

You can organize dropdown lists by using optgroup elements <optgroup>.
This way you can label different groups.

fieldsets

By using fieldsets you create a box around your forms.
You can add a description by using a legend element.
