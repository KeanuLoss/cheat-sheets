Object Oriented Programming

Classes are blueprints to create objects

Objects are instances of classes

Classes have attributes and methods

the attributes define the state / the data that the objects will contain

methods are functions that you're using on the objects

creating a class

1. type " class class-name :"

2. define an initializer function in order to define the attributes of your objects

the attributes need to be stored in a variable named " self.attribute "
example : def __init__ (self, attribute1, attribute2, ...): self.attribute1 = attribute 1, self.attribute2 = attribute2 ...

3. define functions that will be the methods that you can call on objects created from this class
example : def method-name(self): any code that you want to execute

4. create an object using " object-name = class-name(attribute1, attribute2, ...)"

now you've created an object and a method you can use on that object
